# toolhearth.com

Online tools, utilities, calculators, and generators.
Built with vanilla HTML/CSS/JS. Zero dependencies.

## Build and verify

```bash
python3 build.py
python3 seo_audit.py
```

`build.py` regenerates the shared page shell and `sitemap.xml`. The SEO audit checks every generated page for required metadata, valid canonicals and JSON-LD, sitemap integrity, and broken static internal links.

See `LAUNCH_CHECKLIST.md` for deployment, lead-delivery, search indexing, and monetization setup.
